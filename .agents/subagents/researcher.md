# Researcher

Task boundary: answer one external or documentation question.

Required input: question, trusted source preference and decision impact.

Expected output: answer, sources, confidence and remaining unknowns.

Context budget: no whole repository unless needed.

Completion condition: decision can proceed or blocker is explicit.

Non-responsibilities: implementation and final synthesis.

When not to invoke: local repo evidence is sufficient.

Expected length: 10-20 lines.

Model diversity: useful for independent second opinions.

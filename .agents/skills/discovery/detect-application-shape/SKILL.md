# Detect Application Shape

Classify shape independently from framework:

- library
- CLI
- frontend SPA
- monolith or modular monolith
- HTTP service
- Kafka consumer or producer
- stream processor
- scheduled worker
- game
- data pipeline
- agentic application
- hybrid service

Use entrypoints, routes, controllers, listeners, commands, scenes, workers and
package scripts. Do not equate Spring Boot with microservice.

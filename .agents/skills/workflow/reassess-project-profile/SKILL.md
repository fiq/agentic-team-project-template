# Reassess Project Profile

Trigger when runtime, framework, persistence, messaging, deployment, external
integration or project shape changes materially.

Output:

- CHANGE: what changed
- BLAST RADIUS: affected architecture, tests, build, deployment and docs
- MIGRATION PLAN: smallest reversible steps
- QUESTIONS: unresolved material questions

Update `PROJECT_PROFILE.toon` and affected specs, docs and checks.
